Top Signals · 核心事件

1. 微软与 OpenAI 重构合作关系，终结排他性条款

🔴 战略级 📅 2026-04-27 发布
📎 一手来源：Microsoft Official Blog | CNBC 报道 | Ars Technica 分析
微软和 OpenAI 于4月27日联合宣布修订合作协议，核心变化包括：

① 排他性解除：OpenAI 不再绑定 Azure 作为唯一云服务商——微软的 IP 许可证变为非独占性质至2032年，OpenAI 可向任何主流云厂商提供模型服务。
② 收入分成逆转：微软不再向 OpenAI 支付收入分成；反之 OpenAI 向微软支付至2030年（设总上限）。
③ AGI 条款移除：从公开文本中移除通用人工智能触发条件相关条款。
④ 首发权保留：微软仍是 OpenAI "主要云合作伙伴"，OpenAI 产品仍优先在 Azure 首发。
📈 对数据平台的影响：这是自2019年双方建立合作以来最重大的结构性调整。短期影响有限（Azure 仍为首选首发平台），但中长期意味着 OpenAI 模型可通过 AWS/GCP 等其他渠道获取，跨云部署 AI 工作负载成为现实。Databricks 已率先响应——其 Mosaic AI Model Serving 在4月24日即上线 GPT-5.5 托管服务。

2. Google Cloud Next '26 发布 Agentic Data Cloud，数据平台进入 Agent 时代

🔴 产品战略 📅 2026-04-22~24 大会期间发布
📎 一手来源：Google Cloud Blog 总览 | Agentic Data Cloud 详细公告
在 Las Vegas 举办的 Cloud Next '26 上，Google 正式发布 Agentic Data Cloud 战略愿景及产品矩阵：

🏗️ BigQuery 性能与成本：流体自动扩缩容平均降低 34% 查询成本；新增 Measures 语义层（预览版）
🤖 Data Agent Kit（预览版）：在 VS Code / Claude Code / Gemini CLI 等 IDE 中编排数据工程/科学/治理类 Agent
🔧 四大内置专用 Agent：Data Engineering Agent（GA）、Data Science Agent（GA）、Database Observability Agent（预览版）、Deep Research Agent（预览版）
📚 Knowledge Catalog：Dataplex Catalog 演进版，聚合跨平台元数据（SAP/Salesforce/ServiceNow），Gemini 自动补全 Schema
🌐 开放联邦查询：可直接读取 Databricks Unity Catalog、Snowflake Polaris、AWS Glue 目录（均预览版）
🔌 MCP 协议全面支持：BigQuery/Spanner/Looker 等核心引擎提供 MCP 接口
📈 对数据平台的影响：行业头部玩家首次将"Agent Scale"而非"Human Scale"作为架构设计目标。Data Agent Kit 直接对标 Databricks Mosaic AI Agent Framework；开放联邦查询 + MCP 在 Iceberg 标准之上进一步降低跨云门槛。BigQuery 从数据仓库进化为"行动系统"，对 Snowflake/Databricks 构成直接竞争压力。

3. 国家数据局启动 2026 年"数据要素×"大赛，设 17 个赛道含数据基础设施专项

🟡 政策驱动 📅 2026-04-27 发布
📎 来源：国家数据局通知（腾讯新闻转载）
国家数据局联合住建部等部委于4月27日发布通知，启动 2026 年"数据要素×"大赛。

🏆 赛事主题："数据赋能 乘数而上"
📋 赛道设置：16 个行业赛道 + 1 个专业赛道（数据基础设施）
    行业赛道覆盖：工业制造 / 现代农业 / 商贸流通 / 交通运输 / 金融服务 / 科技创新 / 医疗健康 / 医疗保障 / 应急管理 / 气象服务 / 城市治理 / 绿色低碳 / 人力资源 / 体育发展 / 文物保护利用 / 中医药
🏗️ 赛制安排：地方分赛 → 全国总决赛
💡 重点关联：城市治理赛道明确提及 CIM（城市信息模型）平台深度结合，获奖项目可优先纳入住建部 CIM 平台应用案例库
📈 对数据平台的影响："数据基础设施"作为独立赛道首次出现，意味着国家层面开始将数据湖仓/数据编织/数据编织等基础能力视为需重点投入的方向。结合《"数据要素×"三年行动计划》2026 年收尾的时间节点，下半年有望迎来一波数据平台相关的政策性采购高峰。
